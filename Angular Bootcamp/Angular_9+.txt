Commands

- Create new project
  > ng new < app_name >

- Clone or prepare for run an Angular project
  > npm install

- Run angular app nodemon (live-reload) & open browser
  > ng serve -o 

- Add Themes
  * Nebular (SCSS): ng add @nebular/theme
  * Material: ng add @angular/material
  * Bootstrap CDN: 
    - index.html

Routes & Parameters
https://angular.io/guide/router

  - Navigate in HTML Template
    > [routerLink]="['/<route>']" routerLinkActive="active"

  - Navigate from TS component
    > import { Router } from "@angular/router";
      this.router.navigate(["/<route>", "<params?>"]);

  - Retreive Params from route
    > import { ActivatedRoute } from "@angular/router";
      this.activatedRouter.params.subscribe( myParams => {
        // TODO params values in 'myParams' variable
      });
