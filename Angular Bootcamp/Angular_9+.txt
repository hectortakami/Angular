Commands

- Create new project
  > ng new < app_name >

- Clone or prepare for run an Angular project
  > npm install

- Run angular app nodemon (live-reload) & open browser
  > ng serve -o 

- Add Themes
  * Nebular (SCSS): ng add @nebular/theme
  * Material: ng add @angular/material

- Create new component 
  > ng g c < component_name >
